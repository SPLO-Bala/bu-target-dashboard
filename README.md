# BU Target Dashboard

This dashboard reads all changing numbers from `data.csv`.

## Update the dashboard
1. Open `data.csv` in Excel.
2. Update the `current` values for each closer and the Content Target row.
3. Update the date in the `Last Updated` row under the `role` column.
4. Save the file as CSV without changing its name.
5. Commit and push the changes to GitHub.
6. Netlify will automatically redeploy the site.

The Current Deposits total is calculated automatically by adding the four closer totals.
